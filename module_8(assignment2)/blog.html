Scope Basically referes to the area of a variable where it's can be accessed.
Scope of Var:
The scope is global when a var variable is declared outside a function. This means that any variable that is declared with var outside a function block is available for use in the whole window. var is function scoped when it is declared within a function. This means that it is available and can be accessed only within that function.
Scope of Let:
There are some problem with var variable. The let variable comes as an improvement to var declarations.A variable declared in a block with let is only available for use within that block.
Scope of Const:
Variables declared with the const maintain constant values.Const declarations can only be accessed within the block they were declared.A variable declared with const remains the same within its scope. It cannot be updated or re-declared.

NULL:
It is explicitly assigned to a variable to indicate that the variable intentionally does not have a value.
UNDEFINED:
It indicates that a variable has been declared but has not yet been assigned a value.

REST API:
API stands for Application Programming Interface. An API establishes a connection between programs so they can transfer data. REST stands for REpresentational State Transfer. It is a standard that guides the design and development of processes which enable us interact with data stored on a web servers. A REST API (also called a RESTful API or RESTful web API) is an application programming interface (API) that conforms to the design principles of the representational state transfer (REST) architectural style.
